
Plathosys-PRO-Ctrl
Version 1.0.0.1


Mit dem Plathosys-PRO Control Program können Sie folgende Funktionen unserer PRO Produkte steuern:

  ·	Headset
  ·	Schulungsfunktion (mithören)
  ·	Lautsprecher

  
Grundvoraussetzung ist ein angeschlossenes Plathosys Gerät:

Alle Funktionen nutzbar:
   CT-220 PRO / CT-240 PRO / CT-260 PRO
Nur der Lautsprecher ist steuerbar:
   CT-400 PRO / CT-440 PRO / CT-460 PRO


Verwendung des Programms:

Klicken Sie auf die entsprechende Verknüpfung im "Plathosys-PRO" Ordner auf dem Desktop oder verwenden Sie die Verknüpfungen im Startmenü.
Viele finden es hilfreich die benötigten Verknüpfungen in der Windows Taskleiste abzulegen, sodass man immer darauf Zugriff hat und nicht z.B. zunächst eine Anwendung minimieren muss.
Außerdem kann man im Eigenschaften Dialog der Verknüpfung auch eine Tastenkombination zum Aufrufen der Funktion festlegen, z.B. F7 -> Headset einschalten und F8 -> Headset ausschalten.


Deinstallation:

Zum Deinstallieren rufen Sie bitte den Uninstaller im Startmenü auf. Alternativ können Sie natürlich auch den integrierten Programm-Manager von Windows verwenden.
